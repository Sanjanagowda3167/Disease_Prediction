Disease Prediction System
Project Description
This project is a web-based application that predicts diseases such as Diabetes and Heart Disease using Machine Learning models. Users can enter medical parameters through a web interface, and the system provides real-time predictions.

Features
Diabetes prediction using ML model
Heart disease prediction using ML model
User-friendly web interface
Real-time prediction results
Multiple trained models and scalers used
Technologies Used
Python
Flask
Pandas
NumPy
Scikit-learn
HTML, CSS

Step 1: Install Required Libraries

Step 2: Run the Application
py app.py
Step 3: Open in Browser
http://127.0.0.1:5000/
